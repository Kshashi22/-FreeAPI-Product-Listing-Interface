# FreeAPI Product Listing Interface

A clean and responsive product listing interface built with React and Vite for the Web Dev Cohort 2026 assignment.

## Project Description

This project fetches product data from the FreeAPI Random Products API and displays it in a simple e-commerce style layout. Each product card shows the product image, title, brand, category, price, discount, rating, stock, and description.

## API Used

FreeAPI Random Products API:

https://api.freeapi.app/api/v1/public/randomproducts

## Features

- Fetches product data from API
- Displays products in responsive card layout
- Shows product image, price, rating, stock, brand, and category
- Loading state
- Error handling
- Clean modern UI
- Fully responsive design

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- FreeAPI
